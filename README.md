# 🌍 Favourite Places Webpage

A static webpage showcasing my favourite travel destinations, built using HTML and CSS. The project focuses on clean layout design, styling, and user-friendly presentation.

🚀 Features
- Multiple place sections with images and descriptions
- Clean and structured layout
- Simple and attractive UI design
- Beginner-friendly static website

🛠️ Technologies Used
- HTML5
- CSS3

📌 Project Purpose
This project was created to practice frontend development skills such as layout design, styling, and organizing content effectively.

live demo:https://bharathjapathi123.github.io/Favouriteplaces/
